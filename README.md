The is a custom dash design for the GARW IC7.  Learn more about how to insall this dash and discover other dash designs at the GARW discord:  https://discord.gg/wXceyn9BMZ

This dash includes customizations with showing/hiding RPM tach numbers, changing the speed/rpm placements, toggling visibility of various gagues, and enabling a speed/rpm peak guage in any position for more race inspired layouts. Changing the various units for US/EU preferences (MI/KM,PSI/BAR,C/F, MPH & KM/H) is also included.  Please leave feedback for additional features. Enjoy!
