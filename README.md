The is a custom dash design for the GARW IC7.  Come learn more how to insall the dash and find more dash designs at the GARW discord:  https://discord.gg/wXceyn9BMZ

Includes customizations with showing/hiding RPM tach numbers, changing the speed/rpm placements, hiding and showing various gagues, and showing a Peak guage in place of oil temp, etc.
